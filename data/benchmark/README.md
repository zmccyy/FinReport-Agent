# 测试年报 PDF 下载指引

## M2 集成测试需要 3 份 A 股 2024 年报（~200 页/份）

### 来源 1：巨潮资讯网（cninfo.com.cn）

1. 访问 https://www.cninfo.com.cn — A 股上市公司年报官方披露平台
2. 搜索框中输入股票代码，进入「定期报告」→ 选择 2024 年报 PDF 下载

| 股票代码 | 公司 | 行业 |
|---|---|---|
| 600519 | 贵州茅台 | 白酒 |
| 601318 | 中国平安 | 保险 |
| 300750 | 宁德时代 | 新能源电池 |

### 来源 2：上市公司官网「投资者关系」→「定期报告」

### 来源 3：东方财富网（eastmoney.com）

搜索股票代码 → 公告 → 筛选「年报」→ 下载 PDF

## 下载后

将 PDF 放入 `data/benchmark/annual_reports/`：
- `moutai_2024.pdf`
- `pingan_2024.pdf`
- `catl_2024.pdf`

## 合规提示

- 仅用于个人研究测试
- 年报为上市公司公开披露信息
